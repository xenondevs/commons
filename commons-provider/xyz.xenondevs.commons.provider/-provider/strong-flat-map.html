strongFlatMap

abstract fun <R> strongFlatMap(transform: (T) -> Provider<R>): Provider<R>

Creates and returns new Provider that lazily maps to the value of the Provider returned by transform.

Contrary to strongFlatMap, this does not disable lazy evaluation of this. Consequently, updates (for example in subscribers or observers) are only received after a flat-mapped provider has been resolved by evaluating the returned provider, applying transform.

transform should be a pure function.
